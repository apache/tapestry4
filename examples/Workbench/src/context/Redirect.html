This page demonstrates that Tapestry can interoperate with traditional web applications. A listener method may redirect to either an internal resource within the same web application, or to an external URL.

Examples:

  • Forward to internal resource (redirect-target.html)
  • Redirect to external URL (http://jakarta.apache.org/tapestry)

The code for both these cases is very simple: 

    public void redirectInternal(IRequestCycle cycle)
    throws RequestCycleException
    {
        throw new RedirectException("redirect-target.html");
    }

    public void redirectExternal(IRequestCycle cycle)
    throws RequestCycleException
    {
        throw new RedirectException("http://jakarta.apache.org/tapestry");
    }
